All mordern websites and web applications are built using three fundamental terminologies: HTML, CSS and JavaScript. These are the languages of the web.

In this post let's focus on HTML. We will learn what HTML is all about, and why you should learn it

What is HTML?

HTML stands for Hyper Text Markup Language that web developers use to structure the content of a webpage (not a programming language).

HTML consists of elements that describe different types of contents: paragraphs, links headings, images, videos, etc. Web browsers understand HTML and render HTML code as websites

In HTML, each element is made up of three parts:

  1. The opening tag
  2. The closing tag
  3. The actual element

You can lear more at MDN Web Docs

Why Should you learn HTML?

There are countless reasons for learning the fundamental language of web. Here are 5 of them:

  • To be able to use the fundamental web dev language
  • To hand-craft beautiful websites instead of relying on tools like Wordpress or Wix
  • To build web applications
  • To impress friends
  • To have fun
